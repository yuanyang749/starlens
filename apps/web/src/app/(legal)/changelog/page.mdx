## v0.5 — 移动端 AI 会话 UI/UX 重构与 AI Provider 设置对齐

**2026-07**

发布 Starlens 移动端（`@starlens/mobile`）AI 对话及系统设置的全面重构优化，包含以下重大更新：

### 📱 移动端 AI 对话 UI/UX 重构

- **极轻量扁平卡片流** — 移除了臃肿卡片和 1px 实线分割，将会话列表重构为左右窄安全留白（8px）的超清透微阴影扁平卡片，利用率最大化。
- **对话操作菜单溢出显示** — 移除了卡片容器的 `overflow: hidden`，修复了绝对定位的菜单气泡在卡片底端时“删除”按钮被无情截断的 Bug。
- **新建对话马卡龙轻量化** — 弃用原高饱和渐变矩形，改写为扁平无投影马卡龙淡蓝底胶囊新建按钮。
- **顶部栏及搜索栏去框对齐** — 主界面、详情页、对话页的所有 Header 图标去框对齐 32px；主搜索框和下方筛选框统一重构为胶囊圆角，采用无描边扁平淡灰底色，且聚焦时带淡蓝色呼吸发光反馈。
- **原生级 Slide 推入转场** — 左右推入滑出对话视图，退场动画平滑过渡且保留输入框草稿。
- **拟物化半透明底部抽屉** — 重命名与删除对话动作重构为底部托盘，配备金属手柄与毛玻璃背景遮罩。
- **全面屏物理适配与键盘避让** — 视口高度使用 `100dvh`，弹性避让软键盘并自动空出全面屏 Home Indicator 安全区。
- **正文内联仓库名跳转** — 智能匹配 Markdown 里的 `owner/repo` 并渲染为高亮跳转链接，本地收录直达本地，外部项目智能 Fallback 直达 GitHub 官网。

### ⚙️ AI Provider 设置对齐与鉴权

- **系统默认 AI 卡片与使用中状态** — 动态异步读取系统默认模型状态并在列表前展现“系统默认 AI”只读卡片；当前处于激活状态的卡片会有醒目的 `.mobile-badge--active` 蓝色“使用中”高亮徽标。
- **动作按钮扁平化与功能补齐** — 为自定义卡片补齐了 `设为默认` / `取消默认` 动作。所有按钮重写为 34px 扁平无边框马卡龙色系（高亮功能为淡蓝，验证为淡灰，删除为淡红）。
- **敏感模型数据脱敏** — 非 Admin 账号的系统默认模型只透出 “系统默认 AI 已启用” 字段，隐藏具体的 Provider、模型和 baseUrl 等商业敏感细节，Admin 账号则完整显示，对齐 PC 端鉴权脱敏逻辑。

### 🗄️ 执行链步骤条汉化

- 汉化了执行链中复杂查询的英文函数 `run_readonly_query`，映射中文为“执行数据库查询”，并为其绑定了 Lucide 的 `Database` 图标。

## v0.4 — AI 问答全语义覆盖

**2026-06**

新增 8 种 AI 问答意图类型，`ask` 接口支持自然语言问题的全面路由：

- **统计数量（count）** — "我有多少个 Python 项目？"，直接走数据库聚合返回精确数字
- **存在性检查（existence）** — "我有没有收藏 facebook/react？"，快速返回找到或未找到
- **双仓库对比（comparison）** — "langchain 和 llamaindex 哪个更适合 RAG？"，并发拉取详情后 AI 生成结构化对比
- **分布统计（stats）** — "我的收藏按语言怎么分布？"，聚合语言占比、收藏总数和 Star 最多仓库
- **个性化推荐（recommendation）** — "推荐适合初学者的 Python 项目"，基于用户收藏偏好画像给出方向
- **单仓库分析（single_repo）** — "总结一下 vercel/next.js 的用途"，结合 README 摘要进行深度解读
- **条件过滤（structured）** — 支持 star 数区间、收藏时间范围、最近推送时间、备注内容等多维过滤
- **语义搜索（semantic）** — 原有多路全文召回 + AI 重排

搜索接口同步新增过滤字段：`minStars`、`maxStars`、`starredAfter`、`starredBefore`、`pushedAfter`、`hasNote`、`noteContains`。

**2026-06**

CLI 工具（`@starlens-app/cli` v0.1.7）迭代：

- **修复 install-skill 向导 Enter 残留问题** — 多选后回车不再误触下一步

- MCP Server：`search_stars`、`show_star`、`ask_stars` 等工具
- Agent Skill 一键安装，支持 Claude Code、Cursor、Codex、opencode 等客户端
- 用户管理面板与 AI Token 用量追踪（管理员功能）
