# 使用条款

*最后更新：2026 年 6 月*

使用 Starlens 托管服务（`starlens.520ai.xin`）即表示你同意以下条款。如果你使用自托管实例，这些条款不适用。

---

## 服务说明

Starlens 是一个 GitHub Stars 知识管理工具。托管服务以"现状提供"为基础，不保证 100% 可用性。作为个人开源项目，我们尽力维护但不承诺 SLA。

---

## 使用规则

你同意不将本服务用于：

- 大量自动化请求，导致服务对其他用户不可用
- 抓取、反编译或逆向工程服务端逻辑
- 绕过认证或访问其他用户的数据
- 任何违反 [GitHub 服务条款](https://docs.github.com/cn/site-policy/github-terms/github-terms-of-service) 的行为

---

## 账户与数据

- 账户通过 GitHub OAuth 绑定，你对自己的 GitHub 账户安全负责
- 你的 Stars 数据、标签、备注和 Token 是你的个人数据
- 你可以随时申请删除账户数据（见[隐私政策](/privacy)）
- 托管服务不对数据丢失提供赔偿，建议重要备注在本地留存副本

---

- Starlens 源代码遵循 [MIT License](https://github.com/yuanyang749/starlens/blob/main/LICENSE) 开源
- 你收藏的 GitHub 仓库及其内容归原作者所有
- 你在 Starlens 中添加的标签、备注和整理数据属于你自己

---

## 免责声明

- Starlens 仅作为 GitHub Stars 的整理和检索工具，不对任何仓库内容的准确性或安全性负责
- AI 问答功能依赖第三方 AI Provider，回答内容仅供参考，不构成专业建议
- 服务中断、数据迁移或功能变更时，我们会在 GitHub 上提前公告

---

## 条款变更

条款更新时会在 [更新日志](/changelog) 和 GitHub 上说明。继续使用服务视为接受新条款。
