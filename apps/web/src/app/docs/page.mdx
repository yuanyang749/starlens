# Starlens 用户文档

Starlens 是一个面向 GitHub Stars 的个人知识工作台。它帮助你同步收藏仓库、补充标签与备注、使用自然语言搜索，并把收藏过的项目变成可被开发工具调用的上下文。

## 适合谁使用

- GitHub Stars 数量很多，需要更快找回项目的开发者。
- 希望用标签、备注和 AI 摘要整理开源项目的人。
- 希望让 CLI、Agent 或内部工具检索自己收藏仓库的团队。

## 文档内容

- [技术架构](/docs/architecture)：了解 Web 应用、数据库、同步任务和 AI 能力的模块划分。
- [功能说明](/docs/features)：了解仓库搜索、筛选、标签、备注、收藏和 AI 问答。
- [部署方式](/docs/deployment)：了解本地开发、Vercel 部署和自托管部署的基础流程。
